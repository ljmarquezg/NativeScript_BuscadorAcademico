    • 2. Crear espacio de trabajo arrow_drop_down

      1. Crear una carpeta donde se alojará el proyecto en la raíz del disco duro y extraer el contenido del proyecto:
        C:\nativescript\buscador\

      2. Abrir la cónsola de comandos con permisos de administrador

      3. Desde la consola de comandos, navegar hasta la ruta donde alojan los archivos del proyecto. Para este ejemplo se creará el directorio raíz del proyecto en la raíz del disco duro:
        C:\nativescript\buscador

        cd C:\nativescript\buscador

      4. Instalar las dependencias del proyecto ejecutando el siguiente comando:

        tns install

        Espere a que termine el proceso de descargas e instalación

  • 2. Mac OS arrow_drop_down
    • 1. Configurar el servidor arrow_drop_down

      1. Seguir las instrucciones de acuerdo a la documentación de nativescript

        Ir a https://docs.nativescript.org/start/ns-setup-os-x

    • 2. Crear espacio de trabajo arrow_drop_down

      1. Crear una carpeta donde se alojará el proyecto en el escritorio del usuario y extraer el contenido del proyecto dentro de la misma:
        Users/%nombreUsuario%/Desktop/Buscador

      2. Abrir la terminal y navegar hasta la ruta donde alojan los archivos del proyecto. En este ejemplo el escritorio del usuario:
        Users/%nombreUsuario%/Desktop/Buscador

        cd Users/%nombreUsuario%/Desktop/Buscador

      3. Instalar las dependencias del proyecto ejecutando el siguiente comando:

        tns install

        Espere a que termine el proceso de descargas e instalación
    • 3. Ejecutar el proyecto arrow_drop_down

      1. Ejecutar el proyecto utilizando el siguiente comando en cónsola:

        tns run ios

        Esperar a que termine el proceso de ejecución. Si no hay algún dispositivo conectado, automáticamente se abrirá una cónsola de emulador.
