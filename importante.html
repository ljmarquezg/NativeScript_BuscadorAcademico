    • 2. Configurar el servidor arrow_drop_down

      1. Chocolatey

        Ir a https://chocolatey.org/install#install-with-cmdexenpm

        1. Abrir la cónsola de comandos con permisos de administrador

        2. Desde la consola de comandos ejecutar el comando>

          @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"

      2. Instalar nodejs: Desde la consola de comandos ejecutar el comando>:

        choco install nodejs -y

        y esperar a que termine el proceso de instalación.

      3. Instalar JavaSDK8

        choco install jdk8 -y

        y esperar a que termine el proceso de instalación.

      4. Instalar AndroidSDK

        choco install android-sdk -y

        y esperar a que termine el proceso de instalación.

      5. Instalar Android SDK 25, Android SDK Build-tools 25.0.2 o superior, Repositorio soporte Android, repostorio soporte Google.

        %ANDROID_HOME%\tools\bin\sdkmanager platforms;android-25; build-tools;25.0.2 extras;android;m2repository

        y esperar a que termine el proceso de instalación.

    • 5. Errores comunes arrow_drop_down

      1. Mongod. Puerto 8080 en uso:

        Cerrar cualquier otro proceso que está utilizando el puerto 8080 y ejecutar nuevamente el código

        nodemon serve.js

      2. Error comunicación:

        Verificar que los servicio mongod y nodemon serve.js esten corriendo en el servidor

        Una vez iniciados estos procesos presione el botón Reintentar

      3. Error Runtime Error: ev.target closest is not a function

        Este error es un bug del framework Ionic2. Ocurre cuando se intenta utilizar un item slider varias veces sin que termine el proceso de renderizado

        Para solventar el error, se debe refrescar la pantalla del navegador
